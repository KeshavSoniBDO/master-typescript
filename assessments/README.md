# Assessments

This folder contains mastery checks after (or between) modules.

## Files

1. `module-checkpoints.md` - short checkpoint prompts per module
2. `final-capstone.md` - integrated production-style practical
3. `teaching-prompts.md` - instructor prompts for live explanation checks

## How To Use

- Take checkpoint assessments after each module assignment
- Time-box each checkpoint (30-45 min)
- Solve final capstone after Modules 1-7 are complete

## Scoring Focus

Assessments prioritize:

1. quality of mental model
2. boundary safety and runtime awareness
3. type design clarity
4. maintainability of solutions

The goal is durable skill, not memorized syntax.
