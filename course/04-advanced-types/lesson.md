# Module 4: Advanced Types

## Core Mental Model

Advanced types let us transform and compute types from other types.

This is useful when the type should follow the data model instead of being manually duplicated.

## Important Ideas

- Mapped types transform object properties.
- Conditional types choose one type or another.
- Template literal types build string patterns.
- Discriminated unions model state cleanly.
- Exhaustiveness checks protect us when a union grows.

## What To Practice

- Convert repeated object types into mapped types.
- Model states with discriminated unions.
- Use `never` to catch missing cases.
- Read common utility types like `Pick`, `Omit`, `Partial`, and `Record`.
